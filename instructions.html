Implementation Instructions

Quickly set up the self-contained VoidCaptcha overlay on your page.

1. Include the Script

Simply include the VoidCaptcha script before your closing </body> tag: 

<script src="voidcaptcha.js"></script>

The CAPTCHA overlay initializes automatically on page load.

2. Optional Manual Trigger

By default, VoidCaptcha runs automatically, but you can manually trigger it: 

voidCaptcha.init();

Useful if you want to delay CAPTCHA until a certain action.

3. Listen for Verification Events

Detect when the user successfully completes the CAPTCHA: 

document.addEventListener('voidcaptcha-verified', function() {
    console.log('User verified!');
    // Continue with your flow here
});

4. Optional Customization

You can tweak box colors, word length, or cooldowns in voidcaptcha.js: 

const COLORS = ["red", "blue"];      // Box colors
const MIN_WORD_LENGTH = 12;          // Typing word length
const INITIAL_COOLDOWN_MS = 5000;    // Start cooldown
const MAX_TRIES_BEFORE_TIMEOUT = 3;  // Number of attempts before timeout

Important Note

VoidCaptcha is entirely client-side. There is no form connection. Verification is handled internally via the checkbox and overlay.
